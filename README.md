## Run the prototype

No build step. Open `prototype/index.html` in a browser, or serve locally:

```bash
cd prototype
python3 -m http.server 8000   # then visit http://localhost:8000
```

Pages: Home · Courses · Course detail · Weekly Log · About · Contact/Enroll.

## Stack

- **Prototype:** vanilla HTML5 + CSS variables + minimal JS (no framework).
- **Production:** WordPress 6.x (latest), PHP 8.x + MySQL, hosted on GoDaddy.
